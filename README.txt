################################################################## Présentation ################################################################

Tetros est un projet étudiant pour la classe de MP2I 2021-2022 du lycée Claude Fauriel.

Tetros est un clone de Tetris programmé en C. 

Le gameplay est identique au jeu de base, cependant le jeu peut être différent sur divers aspects secondaires.


##################################################################### Jouer ####################################################################

Pour jouer à Tetros, compiler le jeu en exécutant la commmande "make" puis exécuter le fichier "tetros".

Il est recommandé d'utiliser GCC pour la compilation.

La bibliothèque ncurses doit être installée sur la machine pour compiler le jeu.

################################################################## Information ##################################################################

Les fichiers codes sources se trouve dans le dossier "src".

La documentation est disponible dans le dossier "doc".
